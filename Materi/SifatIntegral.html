Integral Tak Tentu Funsi Aljabar dan Sifat-Sifatnya

Pada bagian pertama kita telah mengenal definisi integral dan kaitannya dengan konsep turunan. Nah kali ini kita akan belajar mengenai sifat-sifat yang dimiliki integral tak tentu sehingga kita dapat memahami karakteristik dari integral dalam soal-soal yang akan kita kerjakan.
Bentuk umum integral tak tentu adalah sebagai berikut.
dengan: $c$ adalah konstanta pengintegralan
$f(x)$ adalah integran
$F(x)$adalah fungsi integral umum dan $F(x)$ bersifat $F'(x) = f(x)$
Bagaimanakah aturan-aturan atau sifat-sifat integral tak tentu? Berikut penjelasan mengenai aturan-aturan atau sifat-sifat integral tak tentu fungsi aljabar.

Contoh soal


  1. $\int a dx = ax + c$

    Contoh:

    Carilah antiturunan beri kut:$\int{5}  dx $

    Penyelesaian:

    Dengan menerapkan rumus dasar integral $\int{a}  dx = ax+ c$, maka kita memperoleh:

    $\int{5}  dx$

    $= 5x + c$

    Contoh lain

  2. $\int{x^n} dx = \frac{1}{n+1}x^{n+1} + c, n ≠ -1$

    Contoh:

    Carilah antiturunan berikut: $\int x^{3} dx$

    Penyelesaian:

    Dengan menerapkan rumus dasar integral $\int{x^n} dx = \frac{1}{n+1}x^{n+1} + c, n ≠ -1$, maka kita memperoleh:

    $\int{x^3}  dx$

    $= \frac{1}{3+1}x^{3+1} + c$

    $= \frac{1}{4}x^4 + c $

    Contoh lain

  3. $\int{ax^n} dx = \frac{1}{n+1}x^{x+1} + c, n ≠ -1$

    Contoh:

    Carilah antiturunan berikut: $\int x^{3} dx$

    Penyelesaian:

    Dengan menerapkan rumus dasar integral $\int{ax^n} dx = \frac{1}{n+1}x^{x+1} + c, n ≠ -1$ , maka kita memperoleh:

    $\int{2\sqrt{x}} dx$

    $= \int{2x^{\frac{1}{2}}}$

    $= \frac{3}{\frac{1}{2}+1}x^{\frac{1}{2}+1} + c$

    $= \frac{2}{\frac{3}{2}}x^{\frac{3}{2}} + c$

    $= \frac{4}{3}x^{\frac{3}{2}} +c$

    $= \frac{4}{3}x\sqrt{x}+ c$

    Contoh lain

1.Tentukanlah hasil dari $\int{2x^4\sqrt{x^3}} dx$
Penyelesaian:
Dengan menerapkan sifat integral $\int{a . f(x) dx = a\int{f(x)dx}}$ dan manipulasi aljabar, maka kita memperoleh:

$\int{2x^4\sqrt{x^3}} dx$
= $\int{2x^4 . x^{\frac{3}{2}}dx}$
= $ 2\int{x^4. x^{\frac{3}{2}}dx}$
= $2\int{x^{4+\frac{3}{2}}}$
= $2\int{x^{\frac{11}{2}}}$
= $2[\frac{1}{\frac{11}{2}+1}x^{\frac{11}{2}+1}] + c$
= $2[\frac{1}{\frac{13}{2}+1}x^{\frac{13}{2}+1}] + c$
= $2[\frac{1}{\frac{13}{2}}x^{\frac{13}{2}}] + c$
= $2[\frac{2}{13}x^{\frac{13}{2}}] + c$
= $\frac{4}{13}x^{\frac{13}{2}} + c$

2. Tentukanlah hasil dari $\int{(x+2)^2} dx$
Penyelesaian:
Dengan menerapkan sifat integral $\int{ f(x)+g(x) }dx=\int{f(x)}dx + \int{g(x)}dx$ dan manipulasi alajabar, maka kita memperoleh:
$\int{(x+2)^2} dx$
= $\int{(x+2)(x+2)}$
= $\int{x^2 + 4x + 4} dx$
= $\int{x^2} dx + \int{4x} dx + \int{4}dx$
= $\int{x^2} dx + 4\int{x} dx + \int{4}dx$
= $\frac{1}{2+1} + 4[\frac{1}{1+1}x^{1+1}] + 4x + c $
=$\frac{1}{3}x^3 + \frac{4}{2}x^2 + 4x + c $
=$\frac{1}{3}x^3 + 2x^2 + 4x + c $

3. Tentukanlah hasil dari $\int{\frac{x^3-2x}{\sqrt{x^3}}}dx$
Penyelesaian
Dengan menerapkan sifat integral $\int{ f(x)-g(x) }dx=\int{f(x)}dx - \int{g(x)}dx$ dan manipulasi alajabar, maka kita memperoleh:
$\int{\frac{x^3-2x}{\sqrt{x^3}}}dx$
= $\int{(\frac{x^3}{\sqrt{x}}-\frac{2x}{\sqrt{x}})}dx $
= $(x^3 . x^{-\frac{1}{2}}- 2x. x^{-\frac{1}{2}}) dx$
= $\int(x^{\frac{5}{2}}- 2x^{\frac{1}{2}}) dx $
= $\int{x^{\frac{5}{2}}} dx - \int{2x^{\frac{1}{2}}}dx$
= $\int{x^{\frac{5}{2}}} dx - 2\int{x^{\frac{1}{2}}}dx$
= $\frac{1}{\frac{5}{2}+1}x^{\frac{5}{2}+1} - 2[\frac{1}{\frac{1}{2}+1}x^{\frac{1}{2}+1}] + c$
= $\frac{1}{\frac{7}{2}}x^{\frac{7}{2}+1} - \frac{2}{\frac{3}{2}}x^{\frac{7}{2}} + c$
=$\frac{2}{7}x^{\frac{7}{2}+1} - \frac{4}{3}x^{\frac{7}{2}} + c$
= $\frac{2}{7}x^{3}\sqrt{x} - \frac{4}{3}x\sqrt{x} + c $

4. Carilah nilai $f(x)$ jika $f'(x) = x^3 - 4x^2 + 3$ dan $f(0) = 1$
Penyelesaian
Diketahui : $f'(x) = x^3 - 4x^2 + 3$ , $f(0) = 1$
Ditanya : Carilah nilai $f(x)$ ?
Dijawab : $f'(x) = x^4 - 4x^2$ maka $ f(x) = \int{x^3 - 4x^2 + 3}  dx$
$f(x) = \int x^3 - 4x^2 + 3   dx$
= $\int{x^3} dx - \int{4x^2} dx + \int{3} dx $
= $\int{x^3} dx - 4\int{x^2} dx + \int{3} dx $
= $\frac{1}{3+1}x^{3+1} +4[\frac{1}{2+1}x^{2+1}] + 3x + c$
= $\frac{1}{4}x^{4} - \frac{4}{3}x^{3} + 3x + c$
→ $f(x) = \frac{1}{4}x^{4} - \frac{4}{3}x^{3} + 3x + c$, karena $f(0) = 1$, maka
→ $f(0) = \frac{1}{4}(0)^{4} - \frac{4}{3}(0)^{3} + 3(0) + c$
→ $1 = 0 - + 0 + c $, berarti $c = 1$.
Jadi, nilai $f(x)$ adalah $f(x)=\frac{1}{4}x^{4} - \frac{4}{3}x^{3} + 3x + 1$
